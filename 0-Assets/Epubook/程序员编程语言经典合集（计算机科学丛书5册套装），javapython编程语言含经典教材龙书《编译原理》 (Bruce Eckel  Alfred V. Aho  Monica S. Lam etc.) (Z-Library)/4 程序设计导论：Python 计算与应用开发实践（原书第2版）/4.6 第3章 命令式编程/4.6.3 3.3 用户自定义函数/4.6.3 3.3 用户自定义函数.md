一般而言，一个函数接收零个或者多个输入参数，并返回一个结果。函数的优点之一是可以通过单行语句进行调用来完成实际上需要多行Python语句来完成的任务。更加突出的优点是，通常使用函数的开发人员并不需要知道函数的实现语句。由于开发人员无须知道函数的实现原理，所以函数可以简化程序开发过程。基于上述原因，Python和其他程序设计语言均允许开发人员定义自己的函数。
